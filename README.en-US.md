## 使用
克隆项目到本地
```bash
$ git clone https://github.com/iczer/vue-antd-admin.git
```
安装依赖
```bash
$ yarn install
```
启动
```
$ yarn serve
```
更多信息参考 [使用文档](https://iczer.github.io/vue-antd-admin)
